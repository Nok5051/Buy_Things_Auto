# Selenium 을 이용해 물건 구매하기 
> ## 개요 
> 어느 한적한 오후, 나는 친구가 너무가 가지고 싶어하는 바지가 있길래 발매시간에 맞춰 같이 구매를 도와주기로 했다. 인기가 하늘로 치솟는 브랜드라 나오기만 하면 단 30초만에 품절이 되버리는 브랜드이기에 긴장을 늦출 수 없었다. 6시에 딱 맞춰 클릭에 성공한 친구에게 행복을 줄 수 있단 생각에 기뻤다. 하지만 이게 웬일인가 결제 직전에 품절이 되버려 물건을 구매할 수 없었다. 그렇다 나는 화가 나버렸다. 그래서 Selenium을 이용해 물건을 구매할 수 있는 오토오더(응.. 매크로)를 만들기로 작정했다.
